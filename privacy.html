개인정보처리방침

시행일: 2026년 5월 1일 · 최초 제정

⚠ 본 방침은 정식 출시 전 임시 자가 작성본이며, 법무 검토 완료 후 정식본으로 교체될 예정입니다. 변경 시 본 페이지를 통해 사전 공지합니다.

SkyVault(이하 "본 서비스")는 「개인정보 보호법」 등 관련 법령을 준수하며, 이용자의 개인정보를 보호하기 위해 다음과 같이 개인정보처리방침을 수립·공개합니다.

1. 수집하는 개인정보 항목 및 수집 방법

(1) 이용자 직접 입력 또는 자동 수집

  • 이메일 주소 (사전 알림 신청 또는 회원가입 시)
  • 항공사 마일리지 정보 (이용자가 입력한 항공사명, 보유 마일, 만료일)
  • 마일리지 화면 캡처 이미지 (OCR 자동 입력 기능 사용 시, 일시적으로 처리)
  • 사용자 에이전트(브라우저/앱 정보), 접속 IP, 유입 경로

(2) Gmail 연동 시 추가 수집

  • 항공사 적립/사용 메일 본문 (마일리지 정보 추출 목적, 본 서비스 서버에 저장하지 않음)
  • Google 계정 이메일 주소

2. 개인정보 수집 및 이용 목적

  • 항공사 마일리지 통합 관리 서비스 제공
  • 마일리지 만료 알림 발송
  • 발권 시나리오 추천
  • 서비스 개선을 위한 통계 분석 (개인 식별 불가능한 형태로)
  • 본인 확인 및 부정 이용 방지

3. 개인정보의 처리 위탁

본 서비스는 다음 외부 업체에 일부 정보 처리를 위탁합니다.

위탁 업체위탁 업무보관 위치
Anthropic, PBCOCR 이미지 분석 (Claude Vision API)미국
Supabase, Inc.사용자 데이터 호스팅, 사전 신청 DB한국 (서울 리전)
Google LLCGmail API 연동 (옵트인)미국
Cloudflare, Inc.웹 호스팅, CDN글로벌 분산

4. 개인정보의 국외 이전

OCR 처리 시 마일리지 화면 이미지가 미국 소재 Anthropic 서버로 전송됩니다. 이용자는 OCR 기능 사용 시 이에 동의한 것으로 간주됩니다. Anthropic은 처리 후 30일 내 이미지를 삭제합니다.

5. 개인정보 보유 및 이용 기간

  • 회원 정보: 회원 탈퇴 시까지 (탈퇴 즉시 삭제)
  • 사전 신청자 이메일: 본 서비스 정식 출시 후 1년간 보관 후 파기
  • OCR 이미지: 처리 완료 후 30일 내 삭제
  • Gmail 메일 본문: 본 서비스 서버에 저장하지 않음 (실시간 처리만)
  • 접속 로그: 6개월

6. 이용자의 권리 및 행사 방법

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 개인정보 열람 요구
  • 개인정보 정정·삭제 요구
  • 개인정보 처리 정지 요구
  • 동의 철회

요청은 hello@skyvault.app으로 보내주세요.

7. 만 14세 미만 아동

본 서비스는 만 14세 미만 아동의 회원가입을 받지 않습니다.

8. 안전성 확보 조치

  • Row Level Security(RLS)를 통한 데이터 접근 통제
  • 전송 구간 TLS 1.3 암호화
  • API 키 등 민감 정보의 서버 측 보관
  • 접근 권한 최소화 원칙

9. 개인정보 보호책임자

  • 이메일: hello@skyvault.app

10. 개인정보처리방침 변경

본 방침은 법령, 정책 또는 본 서비스의 내용 변경에 따라 수정될 수 있으며, 변경 시 본 사이트를 통해 사전 공지합니다.
